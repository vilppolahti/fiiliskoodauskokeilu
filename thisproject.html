A collection of simple, well-commented games created as a coding experiment. Each game demonstrates fundamental programming concepts while being fully functional and playable.

Available Games

Snake Game

Classic snake game where you control a growing snake to eat food and avoid collisions.

  • Arrow key controls for smooth movement
  • Score tracking system
  • Screen wrapping for continuous gameplay
  • Self-collision detection

File: snake.py

To run: python snake.py (requires pygame)

Tetris Game

Traditional tetris with all 7 standard tetromino pieces, rotation, and line clearing.

  • All 7 standard tetromino shapes (I, O, T, L, J, S, Z)
  • Piece rotation and movement
  • Line clearing with scoring
  • Game over detection
  • Hard drop functionality

File: tetris.py

To run: python tetris.py (requires pygame)

Chess Game

Complete chess implementation with all standard rules, including check and checkmate detection.

  • All 6 piece types with proper movement rules
  • Check and checkmate detection
  • Turn-based gameplay
  • Standard algebraic notation input
  • Text-based interface for simplicity

File: chess.py

To run: python chess.py

Code Quality

Each game is written with a focus on:

  • Clean, readable code structure
  • Comprehensive comments explaining the why behind design decisions
  • Proper encapsulation and separation of concerns
  • Follows Python best practices
  • Minimal dependencies (only pygame for graphical games)

How to Use

  1. Clone the repository or download the files
  2. Ensure you have Python installed (Python 3.6+ recommended)
  3. For graphical games (Snake, Tetris), install pygame: pip install pygame
  4. Run any game file: python snake.py, python tetris.py, or python chess.py
  5. Read the comments in each file to understand the design decisions
